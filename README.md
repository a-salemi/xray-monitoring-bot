# 🚀 Xray & Shadowsocks Smart Monitoring Bot

یک دستیار هوشمند تلگرامی و سیستم مانیتورینگ خودکار برای مدیریت، تست و بررسی پایداری کانفیگ‌های VPN (پشتیبانی کامل از Vless, Vmess, Trojan و Shadowsocks). 

این پروژه با استفاده از هسته قدرتمند `Xray-core` و ربات تلگرامی پایتون توسعه داده شده است و به شما کمک می‌کند تا سرورهای خود را بدون نیاز به ورود به محیط لینوکس، مستقیماً از طریق تلگرام مدیریت کنید.

## ✨ قابلیت‌ها و ویژگی‌ها

- 🛡 **پشتیبانی از پروتکل‌های مدرن:** پشتیبانی کامل از لینک‌های Vless, Vmess, Trojan و Shadowsocks (شامل تنظیمات پیچیده مانند Reality, WS, gRPC, TLS).
- 📊 **رسم نمودار گرافیکی:** رسم اتوماتیک نمودار میله‌ای از میانگین سرعت و پینگ کانفیگ‌ها.
- 🚷 **سیستم قرنطینه هوشمند:** غیرفعال‌سازی خودکار کانفیگ‌هایی که در ۳ بررسی متوالی قطع باشند.
- ⚡ **تست همزمان (Multi-threading):** بررسی و تست موازی چندین سرور به صورت همزمان بدون افت سرعت.
- 📱 **کنترل پنل تلگرامی:** دکمه‌های شیشه‌ای برای افزودن، حذف، ویرایش، و روشن/خاموش کردن کانفیگ‌ها به صورت زنده.
- 💾 **دیتابیس تاریخچه:** ثبت دائمی سرعت و پینگ‌ها در دیتابیس `SQLite`.

## 🛠 نصب و راه‌اندازی سریع (Easy Install)

برای نصب این سیستم روی سرور اوبونتو/دبیان خود، کافیست دستور زیر را با دسترسی `root` در ترمینال وارد کنید:

```bash
bash <(curl -Ls (https://raw.githubusercontent.com/a-salemi/xray-monitoring-bot/main/install.sh))
```

*(در طول نصب، اسکریپت از شما **توکن ربات تلگرام** و **آیدی عددی اکانت شما** را برای تنظیمات امنیتی درخواست می‌کند).*

## ⚙️ دستورات کاربردی سرویس‌ها

در صورتی که نیاز به متوقف کردن یا بررسی وضعیت سرویس‌ها داشتید:

**بررسی وضعیت ربات:**
```bash
sudo systemctl status ss-bot
```

**بررسی وضعیت سیستم مانیتورینگ:**
```bash
sudo systemctl status ss-monitor
```

**مشاهده لاگ‌های زنده مانیتورینگ:**
```bash
sudo journalctl -u ss-monitor -f
```
